### 원재료(BOM) 탭
- 제품별 **AgCN·PGC 사용량 입력**(50g 생산 기준), 값 입력 시 자동 저장(bom 테이블).
- **월별 소비 리포트**: 발주량 ÷ 50 × 사용량으로 **거래처·품목별 AgCN/PGC 소비** + 합계. BOM 미입력 품목 경고.
- 권한 menu.bom/bom.edit (매니저+ 수정, 일반 보기).

### 모바일 2차 (드로어 내비 + COC 1단)
- 하단 탭바 → **햄버거 그룹 드로어**(현장/데이터/관리). 메뉴 많아도 깔끔, 상단바에 현재 화면명.
- **COC 모바일 1단**: 목록→입력→미리보기 세로 스택. 미리보기 **화면맞춤/실제크기 토글**(PDF는 영향 없음).
- **기록·관리자 표 → 모바일 카드**, 권한 매트릭스 가로 스크롤.

### COC v2 (발행 개선)
- **편집 ↔ 출력 분리**: 입력은 폼, 출력(인쇄/PDF)은 **읽기전용 미리보기**. 입력칸·zoom 제거로 **PDF 깨짐 해결**.
- **합격·불합격 자동판정**: 규격 vs 결과 비교 → 항목별 PASS/FAIL 배지 + 종합판정.
- **발행번호·이력**: 'ORO-YYYY-0001' 자동 부여, 발행일/발행자/버전 기록('발행 확정' 버튼).
- **현미경 이미지 Storage 이전**: DB(base64)→Supabase Storage('coc' 버킷). 로고·도장은 유지.
- **국문/영문 전환** 버튼.

### 주문 직접 추가(수동)
- '주문 가져오기' 탭에 **➕ 주문 직접 추가** 폼: 긴급·이카운트 외 주문을 직접 입력(주문일자/구분/품목/규격/수량/거래처/적요).
- 추가 주문은 목록에 **✋수동** 표시, 생산계획·COC에서 동일하게 사용. 권한(order.import) 필요·감사로그 기록.

### 증빙 원본 보관 + 분기 + PDF 요약본
- **원본 사진 보관**: Supabase Storage 버킷에 영수증 원본 저장(추가 시 자동/수동 첨부). 목록에서 원본 보기·다운로드, **분기 원본 ZIP**. (5년 보관 대비)
- **분기 자동 구분**: 거래일자→1기(1~6월)/2기(7~12월), 분기 필터.
- **PDF 요약본**: 선택 분기 증빙 장부를 A4 PDF로 출력(세무사 전달용). 엑셀과 함께.
- 접대비 선택 시 "누구와/왜" 안내, 원본 '없음' 경고.
- 작업지침서 v2(웹앱 중심 + Storage 원본 + 웍스드라이브 2차백업) 별도 문서.

### COC PDF 저장 + 서식 설정
- **PDF 직접 저장**: 브라우저 인쇄창 대신 성적서를 A4 PDF로 바로 다운로드(html2canvas+jsPDF, 필요할 때만 로드). UI·머리글 안 섞임.
- **인쇄 격리**: 인쇄 시 성적서만 남기고 전부 숨김(하단탭바 포함). '인쇄' 버튼 별도 유지.
- **서식 설정**: 용지(A4/Letter)·여백·로고 높이·글자 배율·푸터(주소/연락처)를 COC에서 조절 → 모든 COC 공통 적용(app_settings.format).
